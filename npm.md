```bash
npm init -y

```
### รัน js แบบไม่ต้องกดปิดเปิดใหม่ ให้ลง `nodemon`

ถ้าอยากรู้ว่าเคยลงในเครื่องยัง ใช้วิธีเช็ค version

```bash
nodemon -v
```

พิมพ์คำสั่งใน terminal เพื่อ install nodemon

```bash
npm install -g nodemon
```

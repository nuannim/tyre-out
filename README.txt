## วิธีการใช้งาน ?


1. double click ที่ไฟล์ run ได้เลยครับ
2. หากขึัน Window defender ให้กด More info 
3. จากนั้นกด Run anyway
จะมีหน้าต่าง Command Prompt แสดงขึ้น โปรดรอสักครู่ เว็บจะทำการเปิดขึ้นมาเองโดยอัตโนมัติ
